# 3.Firebaseで手動デプロイ

一通り動作するアプリケーションができたら、Firebase Hostingを使用してアプリケーションをインターネットに公開してみましょう。

## 3-1.アプリケーションのビルドとデプロイ

アプリケーションをビルドします。
``` bash
npm run build
```

ビルドが正常終了したら、生成したファイルをデプロイします。
```bash
firebase deploy
```

コマンドが正常終了し、`+  Deploy complete!`と表示されていればデプロイは成功です。

Hosting URLが表示されているので、実際にPCやスマートフォン等でアプリケーションにアクセスしてみましょう。

[GitLabのCI/CDを設定する](./04_GitLab_CICDを設定.md)に進んでください。
